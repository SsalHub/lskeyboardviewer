# ⌨️ 로스트사가 키뷰어 (LostSaga Keyboard Viewer)(v1.0)

**로스트사가 전용 키보드 레이아웃 오버레이 및 키보드 세팅 관리 도구입니다.** 기본적인 키뷰어로서의 기능에 더불어, 나만의 키보드 세팅을 정리하고 구성할 수 있습니다.

---

## ✨ 주요 기능

### 1. 실시간 입력 시각화 및 멀티 레이아웃
* **풀 배열(Full)**, **텐키리스(TKL)**, 그리고 화면을 적게 차지하는 **미니멀(Minimal)** 모드를 지원합니다.
* 60칸 그리드 시스템을 적용하여 실제 키보드와 유사한 정밀한 키 배치를 제공합니다.

### 2. 인게임 설정 자동 동기화
* 게임 내의 키보드 설정값을 참조하여 편리하게 키뷰어를 세팅할 수 있습니다.

### 3. 나만의 캐릭터 아이콘 커스터마이징
* **직접 편집 모드**: 각 키에 원하는 용병 아이콘이나 스킬 이미지를 마우스 클릭만으로 바인딩할 수 있습니다.
* **검색 기능**: 수많은 아이콘 중 번호, 이름, 초성 검색을 통해 원하는 이미지를 빠르게 찾을 수 있습니다.

### 4. 키뷰어 커스텀 설정 캡쳐
* 현재 구성한 키뷰어 상태를 선명한 **PNG 이미지**로 저장할 수 있습니다.
* 자신의 키 세팅을 다른 유저와 공유할 수 있습니다.

### 5. 사용자 편의성 극대화
* **윈도우 리사이징**: 창 크기를 조절해도 가로세로 비율이 깨지지 않도록 설계되었습니다.

## 🚀 사용 방법 (How to Use)

### **실행 파일(.exe) 사용 시**
1. `LS_Keyboard_Viewer.exe` 파일을 실행합니다.
2. **반드시 관리자 권한으로 실행**해야 게임 화면 위에서 안정적으로 작동합니다.
3. 우클릭 메뉴에서 `로스트사가 설치 폴더`를 지정한 뒤, 계정을 선택하여 데이터를 불러오세요.

# 필수 라이브러리 설치

# 프로그램 실행
