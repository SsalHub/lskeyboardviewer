# ⌨️ 로스트사가 키뷰어 (LostSaga Keyboard Viewer)

<img width="1042" height="306" alt="곰송" src="https://github.com/user-attachments/assets/a55ff7b7-f28c-4621-8da7-4c92891384d8" />


**로스트사가 전용 키보드 레이아웃 오버레이 및 키보드 세팅 관리 도구입니다.** 기본적인 키뷰어로서의 기능에 더불어, 나만의 키보드 세팅을 정리하고 구성할 수 있습니다.

---

## 🚀 사용 방법

1. `LSKV.zip` 파일 압축을 해제합니다.
2. `LSKeyboardViewer.exe` 파일을 실행합니다.
3. 우클릭 메뉴를 연 뒤, **'용병 아이콘 설정'** 메뉴에서 키보드 설정을 진행합니다.

---

## ‼️주의 사항

1. 관리자 권한 실행 필수
게임 화면 위에 오버레이를 고정하고 키 입력을 정확하게 감지하기 위해, 반드시 **관리자 권한으로 실행**해 주시기 바랍니다.

2. 백신 탐지 관련 안내
본 프로그램은 키보드 입력 감지(hooking) 기술을 사용하므로, 윈도우 디펜더 등 일부 백신에서 바이러스로 **오진**할 수 있습니다.
이는 키뷰어 프로그램의 작동 방식에 따른 자연스러운 현상이며, 사용자의 키 입력 정보나 개인정보를 **절대 수집/전송하지 않으니** 안심하고 예외 설정 후 사용하세요.

3. 저작권 및 수익 창출 관련 안내
본 프로그램은 로스트사가 유저를 위한 **비영리 팬 메이드 툴**입니다.
제작자는 이를 통해 어떠한 수익도 창출하지 않으며, 프로그램에 포함된 이미지 리소스의 모든 저작권은 (주)밸로프 및 원작자에게 있습니다.

### 6. 사용자 편의성 극대화

* **설정 저장 및 불러오기**: 나만의 커스텀 설정을 PC에 저장하고, 언제든지 불러올 수 있습니다.
* 만약 설정이 마음에 들지 않을 경우, 설정 초기화 버튼을 통해 초기 상태로 되돌릴 수 있습니다.
